Meet Azure Synapse, a limitless analytics service that brings together data warehousing and big data analytics

04 Nov 2019, 14:00 by Pradeep

At Ignite 2019, Microsoft revealed a brand-new limitless analytics service called Azure Synapse. Azure Synapse brings together enterprise data warehousing and Big Data analytics. Using this new service, organizations can make use of their data quickly, pulling together insights from all data sources, data warehouses, and big data analytics systems.

Highlights of Azure Synapse:

  • With Azure Synapse, data professionals can query both relational and nonrelational data at petabyte-scale using the familiar SQL language.
  • With Azure Synapse, enabling BI and machine learning is a breeze. It is deeply integrated with Power BI and Azure Machine Learning to greatly expand discovery of insights from all your data and apply machine learning models to all your intelligent apps
  • The Azure Synapse studio provides a unified workspace for data prep, data management, data warehousing, big data, and AI tasks. Data engineers can use a code-free visual environment for managing data pipelines.

Microsoft mentioned the following regarding Azure Synapse launch in the official blog:

Simply put, Azure Synapse is Azure SQL Data Warehouse evolved. We have taken the same industry leading data warehouse to a whole new level of performance and capabilities. Businesses can continue running their existing data warehouse workloads in production today with Azure Synapse and will automatically benefit from the new capabilities when they become generally available.
